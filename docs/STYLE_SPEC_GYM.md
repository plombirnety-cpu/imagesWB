# Спека нового стиля: «Спортзал-мерч / супергерой качалки»

Разбор 3 эталонов владельца (агент их не видел — здесь ДНК стиля словами).

## Общая ДНК (что объединяет все 3 эталона)
1. **Качок-версия узнаваемого персонажа** — тот же персонаж, но с гипертрофированной мускулатурой (грудь/плечи/руки), фронтально, симметрично, голова+торс+руки.
2. **Гантели — фирменный атрибут.** Либо по ТЯЖЁЛОЙ ГАНТЕЛИ В КАЖДОЙ РУКЕ (опущены/на бицепс), либо ДВОЙНОЙ ФЛЕКС (обе руки вверх, гантели у плеч). Ровно 2 гантели, симметрично.
3. **Круглый тематический задник за торсом** — диск с халфтон-точками/энергией/свирлом, ЦВЕТ и характер под персонажа (у Человека-паука золото-халфтон, у Гвен граффити-всплеск, у Рика зелёный портал-свирл). Тёмное кольцо-контур.
4. **Большой АРОЧНЫЙ вордмарк сверху = НАЗВАНИЕ ЗАЛА** — каламбур из имени персонажа/франшизы + gym/фитнес («SPIDER-GYM», «FITNESS-VERSE GYM», «SCIENCE GYM»). Это КРУПНЕЙШИЙ текст.
5. **Слоган-каламбур снизу** — обыгрывает коронную фразу персонажа под фитнес («WITH GREAT WORKOUT COMES GREAT POWER!», «WUBBA LUBBA DUB DUB»).
6. **Бейдж «EST <год>» + город/эмблема** — мелкие акценты (у Паука burst-звезда «Queens NY EST 1962», у Рика атомы + «20·13», у Гвен «QUEENS NYC EST 2014» на брашстроке).
7. **Толстые чёрные контуры, халфтон-тени, ограниченная сочная палитра** (2-4 ярких цвета + крем/контур). Стиль спортивного/винтажного мерча.

## Разбор по эталонам (для вариативности подачи вордмарка)
- **Spider-Gym**: винтажный КОМИКС — золотые буквы с малиновым контуром и кремовой обводкой (шрифт в духе классического лого Spider-Man), маленький паучок на паутинке свисает с буквы, burst-бейдж сбоку. Палитра: малиновый-красный + антик-золото + тёмный сине-зелёный + крем.
- **Spider-Gwen Fitness-verse**: ПАНК/ГРАФФИТИ — рваный дисторшн-шрифт, спрей-текстура, гигантский «GYM» голубым ЗА фигурой, «EST» на брашстроке. Палитра: голубой + маджента/hot-pink + чёрный + белый.
- **Science Gym (Рик)**: КОЛЛЕДЖ/ВАРСИТИ — арочный «SCIENCE» блок-серифом (тёмно-синий с серой обводкой-тенью), огромный «GYM» варсити внизу, атомы-эмблемы, «WUBBA LUBBA DUB DUB» мелко. Палитра: портал-зелёный + navy + серый + офф-уайт.

Вывод: ХАРАКТЕР ШРИФТА/ПАЛИТРУ вордмарка арт-директор подбирает ПОД ФРАНШИЗУ персонажа (комикс / панк-граффити / коллед-варсити / и т.п.), а не один фиксированный шрифт.

## Что создать: ДВЕ записи в docs/STYLE_BANK.json
Формат — как у `34_anime_magazine_cover` (те же поля: id, name_ru, essence, text_treatment, palette_rule, mood_tags, constraints).

### 1) `35_gym_hero_en` — «Спортзал-мерч (качок-супергерой), англ. текст»
Весь текст — АНГЛИЙСКИЙ.

### 2) `36_gym_hero_ru` — «Спортзал-мерч (качок-супергерой), рус. текст»
Весь текст — РУССКИЙ (кириллица). Название зала — русский каламбур («ПАУК-ЗАЛ», «КАЧАЛКА-ВЕРС», «НАУКА-ЗАЛ» и т.п. под персонажа), слоган — русский каламбур коронной фразы, «ОСН. <год>» + город кириллицей.

## Механика полей движка (ВАЖНО для RU-варианта)
- `slogan` в движке САНИТАЙЗИТСЯ до ASCII (`[^A-Za-z0-9 !?'-]`) — кириллица там ВЫРЕЖЕТСЯ. Русский видимый текст должен идти в `quote` (Unicode-safe, чинили ранее).
- `kana`/`name_jp` — ПУСТЫЕ для этого стиля (никакого японского).
- Правило SHIRT-NEUTRAL (нейтральный тон + обводка, ни чёрного, ни белого) применяется кодом ко всему тексту автоматически — но в text_treatment ЯВНО повтори: каждый текстовый блок цветной-из-палитры + толстая контрастная обводка.
- Правило анатомии рук уже глобально: симметричные 2 руки + 2 гантели — ок; явно проси РОВНО две гантели, по одной в руке, без лишних/сросшихся.

## Ограничения (constraints) — обязательно включить
- РОВНО одна качок-фигура + РОВНО 2 гантели (по одной в каждой руке), симметрично; без третьей руки/лишних гантелей.
- Круглый тематический задник за торсом (халфтон/энергия под персонажа), тёмное кольцо-контур; всё на ЧИСТОМ хромакее по краям под вырезку (без бумажной подложки/рамки).
- Толстый чёрный контур + халфтон-тени; ограниченная палитра 2-4 ярких цвета + крем/контур.
- ВЕСЬ текст читается и на ЧЁРНОЙ, и на БЕЛОЙ футболке: нейтральный/акцентный тон + жирный контур; НИКОГДА чисто-чёрный/чисто-белый без контура.
- Название зала — пародийный каламбур, БЕЗ реальных лого брендов/клубов.
- PG, без запрещённой символики.
- RU-вариант: весь текст кириллицей, без латиницы в основных блоках (кроме, при желании, мелкого «EST»→«ОСН.»).
